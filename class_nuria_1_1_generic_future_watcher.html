Detailed Description

The GenericFutureWatcher class can be used to get notified when a Future instances gets finished.

This version can consume a Future with any T. If you want a more specific version use FutureWatcher<>.

Note
Registering a watcher on a already finished future will immediately emit the finished() signals.
